# 🌿 Алгоритм с ветвлением  
**Информатика, 7 класс**

## 🎯 Цель урока

Научиться:
- понимать, как работает условие `if ... else`,
- проверять составное условие с `and`,
- разбирать задания, где программа запускается 9 раз,
- считать, сколько раз программа печатает `ДА`.

---

## 🔍 1. Как выглядит такое задание

На сайте задача устроена так:

1. дана программа вида:

```python
s = int(input())
k = int(input())
if s <= a and k <= b:
    print("ДА")
else:
    print("НЕТ")
```

2. после неё дано **9 пар чисел**;
3. нужно определить, **сколько раз** программа выведет `ДА`.

Значит, нам не нужно запускать программу по-настоящему.  
Нужно просто для каждой пары проверить условие.

---

## 🔍 2. Что означает `and`

Условие:

`s <= a and k <= b`

истинно только тогда, когда одновременно:

1. `s <= a`
2. `k <= b`

Если хотя бы одна часть ложная, программа печатает `НЕТ`.

---

## 🧮 3. Полный разбор задачи формата сайта

### Задача 1

Ниже приведена программа:

```python
s = int(input())
k = int(input())
if s <= 2 and k <= 3:
    print("ДА")
else:
    print("НЕТ")
```

Было проведено 9 запусков программы. При каждом запуске вводились пары чисел:

`(2,3), (4,-10), (1,2), (-3,-3), (-3,5), (1,1), (-2,-3), (10,1), (1,10)`

Сколько раз программа напечатала `ДА`?

---

### Разбор

Условие:

- `s <= 2`
- `k <= 3`

Проверим все пары.

| Пара | `s <= 2` | `k <= 3` | Результат |
|------|----------|----------|-----------|
| `(2,3)` | да | да | ДА |
| `(4,-10)` | нет | да | НЕТ |
| `(1,2)` | да | да | ДА |
| `(-3,-3)` | да | да | ДА |
| `(-3,5)` | да | нет | НЕТ |
| `(1,1)` | да | да | ДА |
| `(-2,-3)` | да | да | ДА |
| `(10,1)` | нет | да | НЕТ |
| `(1,10)` | да | нет | НЕТ |

Сколько раз получилось `ДА`:

`5`

✅ **Ответ: 5**

---

## 🧮 4. Ещё один подробный разбор

### Задача 2

Ниже приведена программа:

```python
s = int(input())
k = int(input())
if s <= 0 and k <= 0:
    print("ДА")
else:
    print("НЕТ")
```

Пары чисел:

`(-1,-1), (0,0), (1,0), (0,1), (-2,-2), (0,-1), (-1,0), (2,2), (-3,1)`

---

### Разбор

Условие:

- `s <= 0`
- `k <= 0`

Проверяем:

| Пара | `s <= 0` | `k <= 0` | Результат |
|------|----------|----------|-----------|
| `(-1,-1)` | да | да | ДА |
| `(0,0)` | да | да | ДА |
| `(1,0)` | нет | да | НЕТ |
| `(0,1)` | да | нет | НЕТ |
| `(-2,-2)` | да | да | ДА |
| `(0,-1)` | да | да | ДА |
| `(-1,0)` | да | да | ДА |
| `(2,2)` | нет | нет | НЕТ |
| `(-3,1)` | да | нет | НЕТ |

Всего `ДА`:

`5`

✅ **Ответ: 5**

---

## ✍️ 5. Как решать такие задания

### Шаг 1. Выпишите оба условия отдельно

Например:
- `s <= 3`
- `k <= 7`

### Шаг 2. Проверяйте каждую пару по двум пунктам

Нельзя смотреть только на одно число.

### Шаг 3. Если обе проверки верны

Ставьте `ДА`.

### Шаг 4. Если хотя бы одна неверна

Ставьте `НЕТ`.

### Шаг 5. В конце посчитайте количество `ДА`

Именно это число и нужно записать в ответ.

---

## 📊 Типичные ошибки

| Ошибка | Как избежать |
|--------|--------------|
| Проверяют только `s`, забывая про `k` | При `and` должны выполниться обе части |
| Путают знак `<` и `<=` | `<=` означает «меньше или равно» |
| Считают число подходящих пар на глаз | Лучше составить маленькую таблицу |
| Считают количество `НЕТ`, а не `ДА` | Всегда читайте последний вопрос задачи |

---

## 💡 Полезный приём

Удобно рядом с каждой парой писать:

- `+`, если будет `ДА`;
- `-`, если будет `НЕТ`.

Потом просто считаются все плюсы.

Это особенно помогает, когда пар 9 и легко запутаться.
