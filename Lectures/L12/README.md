# 📄 Свойства абзаца и страницы в текстовом редакторе  
**Информатика, 7 класс**

## 🎯 Цель урока

Научиться:
- различать свойства абзаца и параметры страницы,
- выбирать правильный вариант для каждого свойства,
- записывать ответ в формате **7 цифр подряд**,
- решать задания этого типа так же, как они даны на сайте.

---

## 🔍 1. Как выглядит такое задание

На сайте задача устроена так:

1. описаны параметры документа;
2. дан список свойств;
3. для каждого свойства есть **4 варианта ответа**;
4. нужно выбрать правильный номер для каждого свойства.

В ответ записывают:

✅ **7 цифр подряд**

по одной цифре для каждого свойства.

### Пример

Если правильные номера такие:

- 1-е свойство → 2
- 2-е свойство → 1
- 3-е свойство → 3
- 4-е свойство → 4
- 5-е свойство → 3
- 6-е свойство → 1
- 7-е свойство → 2

то ответ:

✅ **2134312**

---

## 🔍 2. Какие свойства встречаются

Обычно проверяются:

1. выравнивание абзаца;

3. отступ слева;
4. отступ справа;
5. левое поле страницы;
6. правое поле страницы;
7. верхнее поле страницы.

- левое поле;

## 🧮 4. Полный разбор задачи формата сайта

### Задача 1

В текстовом редакторе создан документ со следующими параметрами:

- выравнивание абзаца: **по ширине**
- отступ первой строки: **1 см**
- отступ слева: **0 см**
- отступ справа: **0 см**
- левое поле страницы: **3 см**
- правое поле страницы: **1.5 см**
- верхнее поле страницы: **2 см**

Для каждого свойства выберите номер правильного параметра.

Пусть варианты такие:

1. **Выравнивание абзаца**  
   `1) По левому краю  2) По правому краю  3) По центру  4) По ширине`

2. **Отступ первой строки**  
   `1) 0 см  2) 1 см  3) 1.5 см  4) 2 см`

3. **Отступ слева**  
   `1) 0 см  2) 0.5 см  3) 1 см  4) 2 см`

4. **Отступ справа**  
   `1) 0 см  2) 0.5 см  3) 1 см  4) 2 см`

5. **Левое поле страницы**  
   `1) 1 см  2) 2 см  3) 3 см  4) 4 см`

6. **Правое поле страницы**  
   `1) 1 см  2) 1.5 см  3) 2 см  4) 3 см`

7. **Верхнее поле страницы**  
   `1) 1 см  2) 1.5 см  3) 2 см  4) 3 см`

---

### Разбор

#### 1. Выравнивание

Нужно `по ширине`.

Это вариант:

✅ `4`

#### 2. Отступ первой строки

Нужно `1 см`.

✅ `2`

#### 3. Отступ слева

Нужно `0 см`.

✅ `1`

#### 4. Отступ справа

Нужно `0 см`.

✅ `1`

#### 5. Левое поле страницы

Нужно `3 см`.

✅ `3`

#### 6. Правое поле страницы

Нужно `1.5 см`.

✅ `2`

#### 7. Верхнее поле страницы

Нужно `2 см`.

✅ `3`

---

### Ответ к задаче 1

Собираем цифры по порядку:

`4 2 1 1 3 2 3`

✅ **Ответ: 4211323**

## ✍️ 6. Как решать такие задания

### Шаг 1. Идите строго по порядку свойств

Не перескакивайте, иначе легко перепутать цифры.

### Шаг 2. Для каждого свойства ищите только один правильный номер

Не нужно решать всё сразу.

### Шаг 3. Сразу записывайте цифру

Лучше делать промежуточную запись:

`4 2 1 1 3 2 3`

### Шаг 4. В конце склейте цифры

Без пробелов.

---

## 📊 Типичные ошибки

| Ошибка | Как избежать |
|--------|--------------|
| Путают поле страницы и отступ абзаца | Поле относится к листу, отступ — к абзацу |
| Путают красную строку и отступ слева | Красная строка — только первая строка абзаца |
| Путают центр и ширину | По ширине текст выровнен и слева, и справа |
| Записывают 7 ответов отдельно | На сайте нужен один код из 7 цифр |

---

## 💡 Самое важное

Эта задача не про вычисления, а про **внимательное сопоставление**.

Нужно не спешить и каждый раз спрашивать:

- это свойство относится к **абзацу** или к **странице**?
- какой **номер** соответствует нужному значению?

Если идти по строкам по порядку, такие задания решаются надёжно.
