# 🖥️ Устройства компьютера: ввод, вывод, хранение  
**Информатика, 7 класс**

## 🎯 Цель урока

Научиться:
- различать устройства ввода, вывода и хранения,
- выбирать нужные устройства из списка,
- записывать ответ в виде номеров по возрастанию,
- решать задания именно в формате сайта.

---

## 🔍 1. Как выглядит такое задание

На сайте задача устроена так:

1. дан список из нескольких устройств;
2. нужно выбрать только те, которые относятся к одной группе:
   - ввод,
   - вывод,
   - хранение;
3. в ответ записываются **номера правильных пунктов** в порядке возрастания.

### Пример

Если верны пункты 1, 3 и 5, то ответ:

✅ **135**

---

## 🔍 2. Что нужно знать

### Устройства ввода

Информация поступает **в компьютер**:
- клавиатура,
- мышь,
- сканер,
- микрофон,
- веб-камера,
- джойстик.

### Устройства вывода

Информация идёт **из компьютера к человеку**:
- монитор,
- принтер,
- проектор,
- колонки,
- наушники,
- телевизор.

### Устройства хранения

Они сохраняют данные:
- SSD,
- жёсткий диск,
- флешка,
- карта памяти,
- оптический диск.

---

## 🧮 3. Полный разбор задачи формата сайта

### Задача 1

Выберите устройства **вывода** информации из предложенного списка:

1. принтер  
2. клавиатура  
3. монитор  
4. микрофон  
5. колонки  
6. сканер  
7. проектор

---

### Разбор

Проверим каждый пункт.

1. `принтер` — вывод  
2. `клавиатура` — ввод  
3. `монитор` — вывод  
4. `микрофон` — ввод  
5. `колонки` — вывод  
6. `сканер` — ввод  
7. `проектор` — вывод

Подходят номера:

`1, 3, 5, 7`

Ответ записываем без пробелов:

✅ **Ответ: 1357**

---

## 🧮 4. Ещё один подробный разбор

### Задача 2

Выберите устройства **хранения** данных из предложенного списка:

1. SSD-диск  
2. процессор  
3. флешка  
4. видеокарта  
5. жёсткий диск  
6. монитор  
7. карта памяти

---

### Разбор

1. `SSD-диск` — хранение  
2. `процессор` — не хранение  
3. `флешка` — хранение  
4. `видеокарта` — не хранение  
5. `жёсткий диск` — хранение  
6. `монитор` — вывод  
7. `карта памяти` — хранение

Подходят:

`1, 3, 5, 7`

✅ **Ответ: 1357**

---

## 🧮 5. Задание на ввод

1. мышь

---

### Разбор

Устройства ввода:

- мышь,
- сканер,

- джойстик.

Это номера:

`1, 3, 5, 6`

✅ **Ответ: 1356**

---

## ✍️ 6. Как решать такие задания

### Шаг 1. Прочитайте, что именно требуется

Нужно выбрать:
- ввод,
- вывод,
- хранение.

### Шаг 2. Проверяйте каждый пункт отдельно

Не ищите ответ целиком сразу.

### Шаг 3. Отмечайте подходящие номера

Например, ставьте `+` напротив правильных.

### Шаг 4. Записывайте номера по возрастанию

Без запятых и пробелов.

---

## 📊 Типичные ошибки

| Ошибка | Как избежать |
|--------|--------------|
| Путают микрофон и колонки | Микрофон вводит, колонки выводят |
| Считают процессор устройством хранения | Процессор только обрабатывает данные |
| Забывают сортировать номера | Ответ пишут по возрастанию |
| Считают оперативную память обычным накопителем | В задачах этого типа берут устройства долговременного хранения |

## 💡 Быстрый вопрос-подсказка

Если сомневаешься, спроси себя:

- информация идёт **в компьютер**?
- информация идёт **из компьютера**?
- данные **сохраняются**?

Ответ на этот вопрос почти всегда сразу даёт правильную категорию.
